# 🚀 Simple Express.js + PostgreSQL CRUD API

This is a simple backend project built using **Node.js, Express.js, and PostgreSQL**.  
It supports basic **CRUD (Create, Read, Update, Delete)** operations for managing users.

---

## 🛠️ Tech Stack

- Node.js
- Express.js
- PostgreSQL
- pg (node-postgres)
- dotenv
